# Sessions & Timers

A session is a single workout from your active plan or a daily WOD. This page explains how to run a session in boostME.

## Starting a session

From the **Home** page, tap the session card to start your next scheduled workout. If you have a session in progress, you'll see a **Resume** prompt instead.

## Session structure

Every session follows a four-phase structure:

```
Warmup → Skill / Strength → Workout → Cooldown
```

Phase tabs at the top of the screen let you see which phase you're in and navigate between them. Not every session has all four phases — some may skip the skill/strength or warmup phase depending on the plan.

## Blocks and exercises

Each phase contains one or more **blocks**. A block groups exercises together with a specific format:

| Block format | How it works |
|---|---|
| **Straight** | Perform each exercise for the prescribed sets and reps, resting between sets |
| **AMRAP** | Cycle through exercises continuously for a fixed time. Score = rounds + extra reps |
| **EMOM** | Complete prescribed work each minute. Remaining time in the minute = rest |
| **For Time** | Complete all prescribed work as fast as possible. Optional time cap |

Each exercise card shows:

- Exercise name
- Sets, reps, or duration
- Weight recommendation (if applicable)
- A video link icon — tap to see a demonstration

## Timers

### Block timers

Timed blocks (AMRAP, EMOM, For Time) have dedicated timers:

- **AMRAP** — Countdown timer with a round counter. Track your completed rounds and extra reps.
- **EMOM** — Shows the current minute and seconds remaining. A voice announcement marks each new minute.
- **For Time** — Count-up timer with an optional time cap. Tap "Finish" when you've completed the work.

### Ramp-up countdown

Before each timed block starts, a configurable countdown gives you time to get in position (default: 5 seconds). You can change this in your [profile settings](../getting-started/profile.md#timer-preferences).

### Set counter

For exercises with reps only (no timer), a tap-based **set counter** lets you track completed sets. Tap the circle to mark a set as done.

### Skip

Use the **X** button to skip an exercise or end a timer early. The timer stops and any pending audio announcements are cancelled.

## Audio announcements

boostME uses pre-recorded voice clips to keep you informed during your workout:

- **Next exercise** — After completing an exercise, the app announces what's coming next (e.g. "Now for Back Squat, 5 sets of 3").
- **Timer cues** — Minute markers during EMOM blocks, countdown warnings during AMRAP.
- **Voice selection** — Choose between a male (Stephen) or female (Tiffany) voice in your profile settings.

!!! tip
    Audio plays as an overlay — it won't interrupt music playing on Spotify or other apps.

## Phase navigation

Use the **Next** and **Back** buttons to move between phases. When you complete the final phase (typically cooldown), the session ends and you're taken to the [Feedback page](feedback.md).
