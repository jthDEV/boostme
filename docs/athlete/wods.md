# Daily WODs & Leaderboard

**WOD** (Workout of the Day) is a daily community workout published by accredited coaches. WODs add a competitive social element to your training.

## Browsing WODs

The **WOD List** page shows today's available workouts. Each WOD card displays:

- Coach name and profile photo
- Workout goal type (AMRAP, For Time, Max Weight, etc.)
- Brief description

WODs can be sorted by coach name. Each WOD is available for 24 hours after publication.

## WOD detail

Tap a WOD to see the full workout breakdown:

- Session goal and target value
- All phases with their blocks and exercises
- Weight recommendations and scaling options

## Completing a WOD

WODs follow the same [session flow](sessions.md) as plan workouts:

1. Tap **Start WOD** on the detail page.
2. Work through each phase (warmup → workout → cooldown).
3. Use the block timers for AMRAP, EMOM, and For Time blocks.
4. Submit your results on the feedback page.

## Recording WOD results

After completing a WOD, you record your result based on the session goal type:

| Goal type | What you record |
|---|---|
| **AMRAP** | Rounds + extra reps |
| **For Time** | Completion time (mm:ss) |
| **Max Weight** | Heaviest lift (kg) |
| **Max Reps** | Total reps |
| **Max Calories** | Total calories |
| **EMOM** | Weight achieved |

## Leaderboard

Each WOD has a leaderboard ranking all athletes who completed it. Rankings are goal-specific:

| Goal type | Ranking |
|---|---|
| **AMRAP** | Most rounds + reps wins |
| **For Time** | Fastest time wins |
| **Max Weight** | Heaviest lift wins |
| **Max Reps** | Most reps wins |
| **Max Calories** | Most calories wins |

The leaderboard shows each athlete's display name and result. A label at the top indicates the ranking method (e.g. "Ranked by: Most Rounds").
