# Managing Athletes

Once you're an active coach with linked athletes, your dashboard becomes your central hub for tracking their progress.

## Dashboard overview

The dashboard displays a grid of your linked athletes. Each athlete card shows:

| Info | Description |
|---|---|
| **Display name** | The athlete's name or initials |
| **Link date** | When they linked to you |
| **Sessions completed** | Number of completed sessions this month |
| **Sessions skipped** | Number of skipped sessions this month |
| **Feedback status** | Whether they've submitted feedback for recent sessions |

## Viewing athlete details

Tap an athlete card to see their full profile and training history:

- Personal information (age, experience level, goals)
- Active training plan overview
- Session history with completion rates
- Recorded weights and performance data

## Link requests

When an athlete requests to link with you:

1. You'll receive a notification.
2. Review the athlete's profile.
3. **Accept** or **decline** the request.

Once accepted, the athlete appears on your dashboard and you can:

- View and edit their training sessions
- Publish WODs they can access
- Interact via in-session chat

## Editing athlete sessions

You can modify individual sessions in an athlete's plan. See [Session Editing](session-editing.md) for details.
