# Session Editing

As a coach, you can edit individual sessions in your linked athletes' training plans. This lets you customize workouts based on an athlete's progress, injuries, or specific needs.

## Accessing session editing

1. Go to your **Dashboard**.
2. Tap an athlete's card to view their profile.
3. Navigate to their active plan.
4. Tap the session you want to edit.

## What you can edit

The session editor gives you full control over the workout structure:

### Phases

Add, remove, or reorder phases:

- **Warmup**
- **Skill / Strength**
- **Workout**
- **Cooldown**

### Blocks

Within each phase you can:

- Add or remove blocks
- Change the **block format** (Straight, AMRAP, EMOM, For Time)
- Set format-specific properties:

| Format | Settings |
|---|---|
| **Straight** | (default — no extra settings) |
| **AMRAP** | Time cap (minutes), label |
| **EMOM** | Time cap (minutes), label |
| **For Time** | Time cap (minutes), label |

### Exercises

For each exercise in a block:

- **Search and add** exercises from the catalog (250+ exercises)
- **Remove** exercises
- Configure **sets**, **reps**, **duration**, and **weight recommendation**

## Exercise browser

The exercise browser lets you search the full boostME exercise catalog:

- Search by name
- Filter by category or equipment
- Each exercise includes video links and scaling options

Tap an exercise in the browser to add it to the current block.

## Saving changes

Changes are saved when you confirm the edit. The athlete will see the updated session the next time they load their plan.

!!! info
    Editing a session only affects that specific session — it does not change the rest of the athlete's plan.
